Питання: На що вказує параметр w при відкритті файлу?

Прізвище: Батраченко
Відповідь: Параметр 'w' при відкритті файлу вказує на те, що файл відкривається для запису. Якщо файл вже існує, вміст файлу буде замінений новим. Якщо файл не існує, він буде створений.

Наступне питання: Які існують режими відкриття файлу у Python?

Прізвище: Фомін
Відповідь: У Python для відкриття файлів існує кілька режимів. Ось найпоширеніші з них:
r (read): Цей режим дозволяє лише читати з файлу. Він починає читати файл з початку. Це режим за замовчуванням.
w (write): Цей режим дозволяє записувати в файл. Якщо файл вже існує, він буде перезаписаний. Якщо файл не існує, створюється новий.
a (append): Цей режим дозволяє додавати дані в кінець файлу. Всі нові дані будуть додаватися в кінець файлу. Файл створюється, якщо він не існує.
b (binary): Цей режим використовується для відкриття файлу в бінарному режимі. Додається до інших режимів, наприклад, rb або wb.
r+ (read/write): Цей режим дозволяє як читати, так і записувати в файл. Файл відкривається для читання та запису з початку.
w+ (write/read): Цей режим дозволяє як записувати, так і читати з файлу. Він перезаписує файл, якщо він існує, або створює новий, якщо файл не існує.
a+ (append/read): Цей режим дозволяє як додавати, так і читати з файлу. Всі нові дані додаються в кінець файлу.
Ці режими можуть бути комбіновані між собою, наприклад, rb, wb, r+, w+, a+.
